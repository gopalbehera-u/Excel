### 📐 Functions

| Function | What it does | Example |
|----------|-------------|---------|
| `SUM` | Adds all values | `=SUM(A1:A10)` |
| `AVERAGE` | Average of values | `=AVERAGE(A1:A10)` |
| `COUNT` | Counts numbers only | `=COUNT(A1:A10)` |
| `COUNTA` | Counts non-empty cells | `=COUNTA(A1:A10)` |
| `COUNTIF` | Count with condition | `=COUNTIF(A1:A10,">100")` |
| `IF` | If condition is true do this else that | `=IF(A1>50,"Pass","Fail")` |
| `Nested IF` | IF inside IF | `=IF(A1>80,"A",IF(A1>60,"B","C"))` |
| `AND` | True only if ALL conditions are true | `=AND(A1>10,A1<100)` |
| `OR` | True if ANY condition is true | `=OR(A1="Yes",B1="Yes")` |
